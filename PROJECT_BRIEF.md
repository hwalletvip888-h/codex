# Project Brief

## 项目定位

Onchain OS 是一个以 Solana 链 meme 币为主的策略研发工作区，先服务于新币发现、风险过滤和流程化执行，再逐步扩展到半自动或自动化。

核心原则：

- 先研究，后执行。
- 先记录风险，再记录收益。
- 先半自动，再考虑全自动。
- 所有私钥、助记词和交易签名必须留在受控环境内。
- SOL meme 策略优先保护本金，不追求每个机会都参与。

## 第一阶段范围

- 整理 OKX Onchain OS 官方资料，优先关注 DEX、token、market、signal、trenches 和 security。
- 建立 Solana meme 新币发现、筛选、执行和复盘模板。
- 建立环境变量和密钥边界。
- 明确哪些步骤可以自动化，哪些步骤必须人工确认。

## 暂不做

- 不在初始化阶段保存私钥或助记词。
- 不默认执行真实交易。
- 不承诺固定收益。
- 不把未经验证的合约加入自动执行名单。
- 不碰无法解释合约风险、流动性风险或卖出风险的 token。
