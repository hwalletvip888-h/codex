# Project Memory

## 当前项目共识

本项目主要针对 Solana 链 meme 币策略，不是泛链上赚币项目。

优先方向：

- SOL 链 meme 新币发现
- 新池子和热门 token 监控
- 风险 token 拦截和过滤
- 观察池、打分、交易计划和复盘
- 后续再考虑半自动或自动化执行

## 关键词解释

当前“阻止”按以下含义处理：

- 阻止高风险 token 进入交易计划
- 阻止疑似貔貅、可疑权限、流动性异常、持仓集中 token 被执行
- 阻止未确认地址、未验证卖出路径、未设置止损的交易

如果后续明确说“狙击”，则理解为：

- 监控新池子和早期成交
- 快速打分和提醒
- 生成交易草案
- 真实买入仍需人工确认，除非另行明确授权

## 主链

第一优先级：

- Solana

暂缓：

- EVM 链 meme
- Base、BSC、Ethereum、Arbitrum 等其他链
- 跨链套利

## 第一版策略重心

优先做：

- SOL meme token 观察池
- Token address 和 mint address 管理
- Pair / pool 地址管理
- 流动性、成交量、持有人、Top holders、dev wallet、bundle / sniper 痕迹
- 是否可卖出和滑点检查
- 风险评分
- 提醒和人工确认流程

暂不做：

- 保存私钥
- 自动广播真实交易
- 自动追高
- 高杠杆或借贷交易
- 未经验证的自动止损修改

## 默认风险态度

Solana meme 币默认视为高波动、高归零概率资产。

默认规则：

- 地址优先于名称和 ticker。
- 先验证能否卖出，再讨论买入。
- 先看流动性质量，再看涨幅。
- 先看持仓分布，再看社群热度。
- 单笔仓位必须小。
- 任何安全项失败，直接跳过。
